üìê How Contour Tracing Works

Algorithm: Marching Squares

Step 1: For each color layer, create a binary mask (1 = this color, 0 = other)

Step 2: Scan for edges where values change (1‚Üí0 or 0‚Üí1)

Step 3: Trace the outline of each region following the edge

Step 4: Simplify the path (remove unnecessary points)

Step 5: Smooth the path (optional - creates bezier curves)

Result: One closed path per distinct region of color

Fine-tune the vectorization settings:

  • simplifyTolerance (0.1-2.0): Higher = fewer points, less detail
  • minAreaThreshold (1-10): Filter out tiny noise regions
  • smoothing (true/false): Enable bezier curve smoothing
